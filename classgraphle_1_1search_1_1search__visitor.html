Detailed Description

template<typename Derived, graph_ref Graph>
class graphle::search::search_visitor< Derived, Graph >

CRTP base class for visitors used with search algorithms. Derived classes may override any of the methods marked with *_base in this class by removing the *_base suffix from the derived method. (i.e. your derived method should be called discover_vertex not discover_vertex_base). Derived methods may return either void or a visitor_result.

Template Parameters
