Detailed Description

template<typename T, bool ReturnCleared = true>
requires requires (T value) { value.clear(); }
class graphle::store::provide_external< T, ReturnCleared >

Storage provider that returns an existing copy of T (owned by the caller) whenever it is invoked, clearing it before each usage with T::clear(), unless ReturnCleared is set to false. The provided storage object must remain valid for the duration of the lifetime of the provide_external provider that it is wrapped in.
