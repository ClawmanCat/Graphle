Detailed Description

template<typename Derived, typename Iterator, typename T>
class graphle::detail::transforming_wrapping_iterator< Derived, Iterator, T >

CRTP base class for (at least forward-) iterators that wrap some other iterator, transforming its value. Unlike wrapping_iterator, this class increments/decrements the iterator automatically by incrementing/decrementing the wrapped iterator.

Derived class should provide the following methods:

  • get(void) const
Template Parameters
DerivedThe CRTP-derived iterator class.
IteratorThe iterator type that is wrapped by Derived.
TThe type returned by get().
