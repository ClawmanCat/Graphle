◆ range_at_key_or_dummy()

template<typename C , typename K , typename P >
constexpr auto graphle::test::range_at_key_or_dummy ( C &  container,
const K &  key,
P &&  projection = std::identity {} 
)
inlineconstexpr

Returns the projected value of the container 'container' at key 'key' if it exists, or the empty range (projection(container.begin())->end(), projection(container.begin())->end()) otherwise. Dumb workaround for some of the test containers that use [set|map].at() to get the edges for a vertex, but that do not have a key for every vertex. This will fail if the provided container is empty, but it never is for our test datastructures.

Parameters
containerA set- or map-like container.
keyThe key to find.
projectionA projection from the containers value type to the stored range type.
Returns
The range projection(*container.at(key)) if the container has the given key or an empty range with the same type otherwise.
