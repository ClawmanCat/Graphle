template<typename Vertex, meta::value_wrapper_of< bool > IsDirected = std::true_type, maybe_vertex_getter< Vertex > GetVertices = meta::none, maybe_edge_getter< Vertex > GetEdges = meta::none, maybe_vertex_edge_getter< Vertex > GetOutEdges = meta::none, maybe_vertex_edge_getter< Vertex > GetInEdges = meta::none, typename Error = decltype(detail::graph_constraints_check<Vertex, IsDirected, GetVertices, GetEdges, GetOutEdges, GetInEdges>())>
struct graphle::graph< Vertex, IsDirected, GetVertices, GetEdges, GetOutEdges, GetInEdges, Error >

Non-owning view of an existing data structure for treating it as a graph.

Note
: For directed graphs a pair [A, B] is considered an edge from A to B and is distinct from an edge [B, A] from B to A. For non-directed graphs, the two are considered equivalent and if the graph has a GetEdges method it should either provide an edge from A to B or from B to A, but not both.
: If the graph is non-directed, the methods GetInEdges and GetOutEdges are considered equivalent, and either or both may be provided.
Template Parameters
VertexThe type of vertex of this graph (The graph will use Vertex* as its vertex type).
IsDirectedTrue if an edge from vertex A to vertex B is the same as an edge from vertex B to vertex A (Parameter is passed as an integral constant, i.e. std::true_type or std::false_type).
GetVerticesThe type of a function object returning the vertices of this graph, or meta::none.
GetEdgesThe type of a function object returning the edges of this graph, or meta::none.
GetOutEdgesThe type of a function object returning the outgoing edges of a vertex, or meta::none.
GetInEdgesThe type of a function object returning the ingoing edges of a vertex, or meta::none.
ErrorWill be set to an error from Graph Constraint Error Types if one of the constraints is not satisfied. This error will show up in any error messages involving the graph's type.
